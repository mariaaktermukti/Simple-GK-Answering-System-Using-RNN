# Simple-GK-Answering-System-Using-RNN

## Overview
This project is a General Knowledge (GK) Question Answering System built using Recurrent Neural Networks (RNN) and Natural Language Processing (NLP) techniques.

The system is capable of understanding user questions and generating appropriate answers from trained data patterns.

---

## Features

- General Knowledge Question Answering
- RNN-based Deep Learning Model
- NLP Text Processing
- Tokenization & Sequence Padding
- Model Training & Prediction
- User Query Handling

---

## Technologies Used

- Python
- TensorFlow 
- NumPy
- Pandas
- Scikit-learn
- NLP
- RNN (Recurrent Neural Network)
- Jupyter Notebook
